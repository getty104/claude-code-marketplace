# Web Search

このスキルは、`gemini` コマンドを使用してWeb検索を実行し、ユーザーの質問に対する最新かつ関連性の高い情報を収集するためのものです。
単純なキーワード検索ではなく、複雑な質問や詳細な情報収集に適しています。

## Instructions

### 1. 検索指示の準備

ユーザーの質問や要求から適切な検索指示を構築します。geminiはLLMツールなので、キーワードではなく自然言語の指示として作成してください

- ユーザーの質問をそのまま、または明確化して指示文として構成
- 何を知りたいのか、何を調べたいのかを明確に記述
- 必要に応じて、回答の形式や詳細度を指定

### 2. 検索の実行

以下のコマンドを使用してWeb検索を実行します

```bash
gemini -p "WebSearch: <調べたい内容や質問を自然言語で記述>" --yolo
```

- `-p`: LLMへの指示を指定するプロンプトオプション
- `--yolo`: 確認なしで即座に実行

### 3. 結果の処理

検索結果を確認し、ユーザーの質問に対する回答を構築します

- 関連性の高い情報を抽出
- 必要に応じて複数の検索結果を統合
- 情報源を明記
- 検索結果が不十分な場合は、異なるクエリで再検索を検討

### 4. 回答の提供

検索結果を基に、ユーザーに対して明確で分かりやすい回答を提供します。
