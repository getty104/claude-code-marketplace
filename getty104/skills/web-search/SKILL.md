# Web Search

このスキルは、`gemini` コマンドを使用してWeb検索を実行し、ユーザーの質問に対する最新かつ関連性の高い情報を収集するためのものです。
単純なキーワード検索ではなく、複雑な質問や詳細な情報収集に適しています。

## Instructions
scripts/web-search.sh スクリプトは、指定されたクエリに基づいてWeb検索を実行し、関連する情報を収集します。
このスクリプトを実行することで、ユーザーの質問に対する包括的な回答を構築できます。
検索結果を確認し、ユーザーの質問に対する回答を構築します。

- 関連性の高い情報を抽出
- 必要に応じて複数の検索結果を統合
- 情報源を明記
- 検索結果が不十分な場合は、異なるクエリで再検索を検討
